# Learning-Linux

```pwd```              >> Print the current working directory

`cd x`              >> Change the current directory to x

`cd ~`              >> Change to the home directory

`cd ..`             >> Move one directory up

cd ../..          >> Move two directories up

ls x              >> List files or directories named x

ls -a x           >> List all files, including hidden ones, in directory x

ls -l x           >> List files/directories in x with detailed information

ls -la x          >> List all files (including hidden) in x with detailed information

ls -R x           >> List directory x and all subdirectories recursively

ls x\*             >> List files or directories starting with x

ls \*x             >> List files or directories ending with x

ls | grep x       >> Search for files or directories containing x in their name

touch x           >> Create an empty file named x

nano x            >> Open file x in the nano text editor

cat x             >> Display the contents of file x

rm x              >> Delete file x

rmdir x           >> Delete an empty directory named x

rm -rf x          >> Delete directory x and all its contents (forcefully)

mv a b            >> Rename or move file/directory a to b

cp a b            >> Copy file a to b

cp -r a b         >> Copy directory a to b recursively

echo x            >> Print the text x

echo $x           >> Print the value of variable x

free              >> Display memory and swap usage

lscpu             >> Display CPU information

lsmem             >> Display memory information

whatis x          >> Show a brief description of command x

whereis x         >> Show the location of command x binaries and sources

man x             >> Display the manual page for command x

info x            >> Display the info documentation for command x

x --version       >> Show version of package or command x

history           >> Show command history

history number    >> Show the last number commands from history

history -c        >> Clear command history

export sina=ls    >> Create an environment variable named sina with value ls

sudo adduser x    >> Add a new user named x

sudo deluser x    >> Delete user x

su - x            >> Switch to user x

who               >> Show who is currently logged in

apt search x         >> Search for packages named x

sudo apt install x   >> Install package x

sudo apt remove x    >> Remove package x

sudo apt update      >> Update the package list

sudo apt upgrade     >> Upgrade installed packages

snap                 >> Snap package manager

sudo snap install x  >> Install snap package x
